## Imagen OG (Open Graph)

Crear un archivo `og-image.png` en esta carpeta con las siguientes especificaciones:

- **Dimensiones:** 1200 × 630 px
- **Formato:** PNG o JPG
- **Contenido sugerido:** Logo JCG Info Tech centrado sobre fondo oscuro (#080c10),
  con el tagline "Soluciones Tecnológicas" y el color de acento cyan (#00d2b4).

Esta imagen aparece cuando se comparte el sitio en WhatsApp, LinkedIn, Facebook, etc.
Sin este archivo las redes sociales usarán una miniatura genérica.
